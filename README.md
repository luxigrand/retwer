# Periodic Table Guessing Game

An interactive periodic table guessing game with dark theme and bilingual support (Turkish/English).

## Features

- **Interactive Periodic Table**: Click on any element to start a guessing game
- **Multiple Choice Questions**: Each element shows 4 options (1 correct + 3 random wrong answers)
- **Score Tracking**: Track your correct and total answers
- **Bilingual Support**: Switch between Turkish and English languages
- **Dark Theme**: Beautiful dark theme with color-coded element groups
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Play

1. Open `index.html` in a web browser
2. Click on any element in the periodic table
3. A modal will appear showing the element's symbol and atomic number
4. Choose the correct element name from 4 multiple choice options
5. Get instant feedback and see your score update
6. Use the language toggle (EN/TR) to switch between English and Turkish

## Element Categories

Elements are color-coded by their categories:
- Alkali Metals
- Alkaline Earth Metals
- Transition Metals
- Post-transition Metals
- Metalloids
- Nonmetals
- Halogens
- Noble Gases
- Lanthanides
- Actinides
